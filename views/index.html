    • About

  • GET The following GET endpoints are provided:
    /notes - to get all notes
    /notes/:id - to get note by id
    /notes/stats - to get statistics about notes
  • POST The following POST endpoint is provided:
    /notes - to post new item

    You should provide the following body:
    name: length > 3 symbols
    date: string, length = 10 symbols
    category: string, 25 > length > 3 symbols
    content: length > 3 symbols
    dates: string
    status: number (1 or 0)

    It returns added item as a result
  • DELETE The following DELETE endpoint is provided:
    /notes/:id - to remove item

    It returns deleted item
  • PATCH The following PATCH endpoint is provided:
    /notes/:id - to remove item

    You should provide the following body:
    name: length > 3 symbols
    date: string, length = 10 symbols
    category: string, 25 > length > 3 symbols
    content: length > 3 symbols
    dates: string
    status: number (1 or 0)

    It returns changed item as a result
